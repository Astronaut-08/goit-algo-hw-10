З проведених експериментів та наведених даних, з використанням бібліотеки scipy для обчислення інтеграла
ми наглядно бачимо, що обчислення інтеграла методом Монте Карло та функції quad близькі один до одного 

функція обчислює інтеграл за визначеним алгоритмом що приводить до отримання одного й того ж результату щоразу
метод Монте Карло має інший підхід, він екпериментально обчислює інтеграл щоразу викорстовуючи різні рандомізовані дані 
для отримання максимально наближеного результату, що можна побачити зі зібльшенням кількості експериментів

функція squad стабільно при наших умовах видає: 2.666666666666667
Монте карло при 10 експериментах видає:         2.67688
Та якщо збільшити експерименти до 10_000:       2.6659503999999985

Висновок, метод Монте Карло дає максимально наближений результат при проведенні великої кількості експериментів, але не точний
враховуючи вищенаведене він може бути дуже корисний в ситуаціях де точний результат визначити неможливо
